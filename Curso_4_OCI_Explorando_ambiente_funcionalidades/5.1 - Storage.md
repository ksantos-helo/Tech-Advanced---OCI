# ☁️ OCI Oracle Certification - Storage (Resumo para Estudos)

---

# 💾 O que é Storage na OCI?

O **OCI Storage** é um serviço de:

- 📦 Armazenamento de dados
- 🔐 Seguro e escalável
- ⚡ Alto desempenho
- 🌍 Acesso rápido e distribuído

---

# ❌ Problemas do armazenamento local (On-Premise)

- 💸 Alto custo de infraestrutura
- 🧱 Necessidade de backups duplicados (10TB → 20TB → 30TB)
- 🐢 Acesso lento a dados antigos
- 📉 Baixa escalabilidade

---

# 🧠 Conceitos importantes

## 📌 Tipos de armazenamento

- 🔒 Persistência → dados continuam existindo
- ⏳ Durabilidade → tempo de retenção dos dados
- ⚡ Performance → velocidade de acesso
- 📦 Capacidade → volume de dados suportado

---

## 🌐 Conectividade

- 🖥️ Local Storage
- 🌍 Network Storage

---

# 💾 Tipos de Storage na OCI

---

## ⚡ Local NVMe

- 🔥 Alta performance
- 🖥️ Disco local da instância
- ⚠️ Não é persistente
- 🚀 Ideal para workloads extremamente rápidos

---

## 💾 Block Volume

- 🔗 Conectado via rede
- 🔒 Persistente e durável
- 💻 Funciona como HD/SSD externo
- 📌 Usado em instâncias (VMs)

---

## 📁 File Storage

- 📂 Sistema de arquivos compartilhado
- 👥 Acesso por múltiplas instâncias
- 📌 Baseado em pastas e arquivos (NFS)

---

## ☁️ Object Storage

- 🌐 Acesso via internet
- 📦 Ideal para:
  - imagens
  - vídeos
  - backups
- 🔥 Altamente escalável

---

# 🚚 Migração de dados

## 📦 Data Transfer Disk
- Envio físico de discos para OCI

## 🧳 Data Transfer Appliance
- Dispositivo físico para grandes volumes de dados

## 🌐 Storage Gateway
- Servidor Linux para migração de dados para OCI

---

# 🎯 O que cai na prova

✅ Block Volume = disco de instância (persistente)  
✅ Object Storage = arquivos web (imagens, vídeos)  
✅ File Storage = compartilhamento entre instâncias  
✅ Local NVMe = alta performance, não persistente  
✅ Storage Gateway = migração de dados  
✅ Data Transfer = envio de grandes volumes  

---

# 🧠 Dicas de certificação

- 🖥️ VM precisa de disco → Block Volume  
- 🌐 Arquivo web → Object Storage  
- 👥 Compartilhamento → File Storage  
- ⚡ Performance extrema → Local NVMe  

---

# 🏆 Resumo final

O **OCI Storage** oferece diferentes tipos de armazenamento para cada necessidade, combinando **performance, escalabilidade, segurança e flexibilidade**, permitindo escolher a melhor solução para cada tipo de aplicação. 🚀
