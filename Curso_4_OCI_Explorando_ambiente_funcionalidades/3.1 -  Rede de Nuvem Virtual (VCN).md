# 🌐 OCI Virtual Cloud Network (VCN)

## 📌 O que é VCN?

A **VCN (Virtual Cloud Network)** é a rede virtual da OCI.

Ela permite criar e gerenciar redes:

- 🌍 Públicas
- 🔒 Privadas
- ☁️ Isoladas na nuvem
- ⚙️ Totalmente configuráveis

📌 É uma rede definida por software (SDN), não física.

---

# 🚀 Benefícios da VCN

- ⚡ Baixa latência
- 📈 Escalabilidade
- 🛡️ Alta disponibilidade
- 🔐 Segurança de rede
- 🌐 Integração híbrida e multicloud
- 🎯 Controle total de tráfego

---

# ⚠️ Problemas sem VCN

- 🚫 Acesso não autorizado
- 🔓 Falhas de segurança
- 🐢 Alto tempo de resposta
- 🏢 Dificuldade em ambiente on-premises
- 🔗 Integração complexa com outras nuvens
- 📉 Escalabilidade limitada

---

# 🧩 Estrutura de Rede

## 📍 CIDR Block

Faixa de IPs da rede.

### Exemplo:

```text
10.0.0.0/16
```

# 🌍 Subnets
Divisões internas da VCN.

##  🌐 Subnet Pública
Permite acesso externo.
```
10.0.0.0/24
```

## 🔒 Subnet Privada
Sem acesso direto da internet.
```
10.0.1.0/24
```

# 🚪 Gateways na OCI
##  🌍 Internet Gateway
Permite acesso direto à internet.

➡️ Entrada e saída.

##  🔄 NAT Gateway
Permite apenas saída para internet.

➡️ Recursos internos acessam fora, mas ninguém entra.

##  ☁️ Service Gateway
Permite acesso privado a serviços públicos da OCI.

Exemplo:
Object Storage

Outros serviços Oracle

##  🔀 Dynamic Routing Gateway (DRG)
Conecta VCN com:

🏢 On-Premises

🔗 Outras nuvens

🌍 Redes externas
