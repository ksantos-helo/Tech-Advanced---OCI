# 🔐 OCI Security — Resumo

## 📌 O que é o módulo de Security?
O módulo de **Security da OCI** reúne serviços e práticas para proteger:

- 🔒 Dados  
- 🖥️ Infraestrutura  
- 🌐 Aplicações  

---

## ⚠️ Principais problemas de segurança

- 🚫 Acessos não autorizados  
- 📤 Vazamento de dados (permissões excessivas)  
- 🌐 Ataques DDoS  
- 🐞 Vulnerabilidades em aplicações  
- ❌ Falta de monitoramento e resposta a incidentes  
- 📜 Não conformidade (ex: LGPD)

---

## ⚖️ Modelo de Segurança Compartilhada

### ☁️ Responsabilidade da Oracle
- Data centers físicos  
- Rede física  
- Infraestrutura e virtualização  

### 👤 Responsabilidade do cliente
- 🔐 Dados  
- 🧑‍💻 IAM (acessos e permissões)  
- 💻 Sistemas operacionais  
- 📱 Dispositivos (endpoints)

---

## 🧱 Segurança em Camadas

### 🌐 1. Proteção de rede e aplicações
- 🛡️ **WAF (Web Application Firewall)**
  - Protege contra ataques web e DDoS
- 🔥 **Network Firewall**
  - Monitora tráfego e atividades suspeitas

---

### 👤 2. Identidade e acesso
- 🔑 **IAM (Identity and Access Management)**
  - Define quem acessa o quê
- 🔐 **MFA (Autenticação multifator)**

---

### 🖥️ 3. Sistema operacional e workloads
- 🛡️ Shield (proteção avançada de VM)
- 🧱 Host dedicado (isolamento)
- 🔄 OS Management (patches e updates)

---

### 🔐 4. Proteção de dados
- 🔑 **Vault**
  - Gerencia chaves de criptografia
- 📜 Certificados SSL

---

### 🔍 5. Detecção e resposta
- 🧠 **Cloud Guard**
  - Detecta riscos e configurações erradas
- 🚫 **Security Zones**
  - Impõem regras de segurança (ex: sem acesso público)

---

### 🔗 6. Monitoramento e integração
- 📊 Auditoria contínua  
- 🔎 Varredura de vulnerabilidades  
- 🔄 Integração entre serviços  

---

## 🧠 Exemplo prático

- Problema: ex-funcionário com acesso ativo  
- Risco: vazamento de dados  
- Solução OCI:
  - IAM + políticas de acesso  
  - Monitoramento com Cloud Guard  

---

# 🎯 O QUE MAIS CAI NA CERTIFICAÇÃO OCI

## 🔥 Pontos-chave (memorizar!)

- ⚖️ **Modelo compartilhado**
  - Oracle → infraestrutura
  - Cliente → dados + acesso
- 🔐 **IAM = controle de acesso (quem pode o quê)**
- 🔑 **MFA aumenta segurança**
- 🛡️ **WAF = proteção contra ataques web/DDoS**
- 🔥 **Network Firewall = controle de tráfego**
- 🔑 **Vault = gerenciamento de chaves**
- 🧠 **Cloud Guard = monitoramento contínuo**
- 🚫 **Security Zones = regras rígidas de segurança**
- 📊 Auditoria e monitoramento são contínuos

---

## 💡 Dica de prova
Se a questão falar:
- "quem acessa o quê" → **IAM**
- "proteção contra ataques web" → **WAF**
- "monitoramento inteligente de segurança" → **Cloud Guard**
- "gerenciamento de chaves" → **Vault**
- "modelo de responsabilidade" → **Shared Responsibility Model**
