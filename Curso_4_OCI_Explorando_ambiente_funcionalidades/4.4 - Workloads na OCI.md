# ☁️ OCI Oracle Certification - Container Workloads (Resumo para Estudos)

---

# 📦 O que são Container Workloads na OCI?

Container Workloads permitem **executar aplicações em containers** de forma:

- ⚡ Leve
- 🔄 Portátil
- 📈 Escalável

👉 Containers empacotam:
- aplicação + dependências

---

# 🚀 Serviços de Containers na OCI

## ☸️ OKE (Oracle Kubernetes Engine)
- Orquestra containers
- Escala automaticamente
- Kubernetes gerenciado

---

## 📦 Container Instances
- Executa containers **sem gerenciar servidores**
- Alternativa ao Kubernetes

---

## 🗂️ Container Registry (OCIR)
- Repositório privado de imagens
- 🔒 Seguro e versionado

---

## 🎯 Diferencial OCI

- 💰 Sem custo para gerenciamento do Kubernetes
- 💵 Paga apenas pelos recursos usados

---

# ❌ Problemas comuns com containers

- 🧩 Complexidade na orquestração
- 🐢 Alta latência e baixa performance
- ⚙️ Falta de automação
- 🔒 Segurança de imagens
- 📉 Dificuldade de escalar
