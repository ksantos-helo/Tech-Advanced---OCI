# 🔐 OCI Vault (Cofre) — Resumo

## 📌 O que é o Vault?
O **Vault** é um serviço da OCI para **gerenciar chaves criptográficas e segredos** com segurança.

- 🔑 Armazena:
  - Chaves de criptografia  
  - Senhas  
  - Tokens  
  - Certificados  
  - Chaves SSH  

- 🚫 Evita guardar segredos em:
  - Código
  - Arquivos de configuração

---

## 🔒 Modos de proteção

### 💻 Software-based
- Chaves armazenadas em servidor
- Podem ser exportadas

### 🧱 HSM (Hardware Security Module)
- Chaves em hardware dedicado
- ❌ Não exportáveis
- 🛡️ Alta segurança (FIPS 140)

---

## ⚙️ Algoritmos suportados

- 🔐 **AES** (simétrico)
- 🔑 **RSA** (assimétrico)
- ✍️ **ECDSA** (assinatura digital)

---

## 🔄 Rotação de chaves

- Permite **trocar chaves periodicamente**
- Aumenta a segurança

---

## 🧠 Criptografia em camadas (Envelope Encryption)

- 🔑 **DEK (Data Encryption Key)**
  - Criptografa os dados

- 🔐 **Master Key**
  - Criptografa a DEK

➡️ Fluxo:
1. Dados → criptografados com DEK  
2. DEK → criptografada com Master Key  

---

## 🔍 Funcionamento (resumo)

### 📤 Criptografando
1. OCI solicita DEK ao Vault  
2. Vault retorna DEK criptografada  
3. Dados são criptografados  
4. Dados + DEK criptografada são armazenados  

### 📥 Descriptografando
1. OCI envia DEK criptografada ao Vault  
2. Vault descriptografa com Master Key  
3. Dados são descriptografados  

---

## ⚠️ Atenção importante

- ❌ Se a **Master Key for deletada**:
  - 🚫 Dados não podem ser recuperados

---

## 🔐 Acesso e segurança

- Controlado via:
  - 🔑 IAM (políticas)
  - 📊 Auditoria (logs)

---

## ⚙️ Uso prático (resumo)

1. Criar Vault  
2. Criar Master Key  
3. Criar política IAM (permitir uso da chave)  
4. Usar chave em serviços (ex: Object Storage)  

---

## 💾 Integração

- Pode ser usado com:
  - Object Storage
  - Block Volume
  - Outros serviços OCI

---

# 🎯 O QUE MAIS CAI NA CERTIFICAÇÃO OCI

## 🔥 Pontos-chave (memorizar!)

- 🔐 **Vault = gerenciamento de chaves e segredos**
- 🧱 **HSM = alta segurança (não exportável)**
- 🔑 **Master Key protege a DEK**
- 🧠 **Envelope Encryption (camadas)**
- ⚠️ **Se perder a Master Key → perde os dados**
- 🔄 **Rotação de chaves aumenta segurança**
- 🔑 Controle via **IAM policies**
- ☁️ Integra com Object Storage (customer-managed keys)

---

## 💡 Dica de prova
Se a questão falar:
- "armazenar segredos com segurança" → **Vault**
- "criptografia com chave do cliente" → **Customer-managed keys (Vault)**
- "DEK + Master Key" → **Envelope Encryption**
- "perda da chave impede acesso aos dados" → **Vault**
