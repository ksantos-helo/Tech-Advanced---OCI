# 📁 OCI File Storage — Resumo

## 📌 O que é File Storage?
O **File Storage** é um serviço de armazenamento baseado em **sistema de arquivos hierárquico**, semelhante a pastas e diretórios que usamos no dia a dia (Windows/Linux).

- Organizado em:
  - 📂 Diretórios
  - 📄 Arquivos
- Funciona como um **filesystem compartilhado na nuvem**

---

## 🔗 Protocolos suportados

- 🐧 **NFS (Network File System)** → Linux
- 🪟 **SMB (Server Message Block)** → Windows

---

## 🎯 Casos de uso

- 🏢 Aplicações empresariais (ex: E-Business Suite)
- 👥 Compartilhamento de arquivos entre equipes
- ⚙️ Microsserviços e containers
- 🚀 HPC (High Performance Computing)
- 📊 Big Data e Analytics

---

## 🧠 Como funciona

- Sistema de arquivos hospedado na nuvem (OCI)
- Conectado a várias instâncias de computação
- Permite:
  - 🔄 Leitura e escrita simultânea
  - 👥 Compartilhamento entre múltiplas VMs

---

## 🛡️ Segurança

- 🔐 Criptografia:
  - Em repouso
  - Em trânsito
- 💾 Snapshots:
  - Backups automáticos
  - Recuperação de dados (ex: exclusão acidental)

---

## 🔧 Como usar

1. Criar o File System
2. Montar nas instâncias (mount)
3. Utilizar como diretório normal

---

## ⚡ Característica chave

- 🔥 **Múltiplas instâncias acessam ao mesmo tempo**
- Ideal para ambientes colaborativos e distribuídos

---

- 📁 File Storage = **sistema de arquivos (tipo pasta compartilhada)**
