# ☁️ Multicloud e Nuvem Distribuída na OCI

## 📌 Problema: Flexibilidade na Nuvem

Empresas enfrentam desafios ao usar nuvem e infraestrutura local:

- ⚖️ Restrições de conformidade (LGPD e regulações)
- 🌍 Limitações para armazenar dados fora do país
- 🐢 Problemas de latência
- 💸 Custos variáveis
- 📉 Desempenho inconsistente
- 🔗 Integração entre ambientes locais e outras nuvens

### Exemplo:

🏛️ Órgãos públicos (prefeituras, governos) não podem armazenar dados sensíveis em outros países.

📌 No Brasil, dados devem permanecer em regiões nacionais como:

- 📍 São Paulo
- 📍 Vinhedo

---

# ☁️ Tipos de Nuvem OCI

## 🌎 OCI Public Cloud – Nuvem Pública

Infraestrutura global compartilhada.

### Benefícios:

- 🌍 +50 regiões globais
- 📈 Escalabilidade
- 💰 Custo-benefício
- 🛡️ Alta disponibilidade

---

## 🏢 OCI Dedicated Region – Região Dedicada

Área exclusiva do datacenter Oracle para um cliente.

### Benefícios:

- 🔒 Isolamento físico total
- 🌐 Rede dedicada
- 🖥️ Hardware exclusivo
- ⚖️ Ideal para regras rígidas de compliance

---

## 🛠️ Oracle Alloy – Cloud Personalizada

Nuvem personalizada criada com parceiros.

### Exemplo:

- 📡 Empresas de telecomunicações
- 🏢 Provedores de serviços

---

## 🏠 Oracle Cloud@Customer – Nuvem no Cliente

Serviços OCI rodando dentro do datacenter do cliente.

### Benefícios:

- 🏢 Mantém infraestrutura local
- ☁️ Recursos de nuvem no ambiente interno
- 🔄 Modelo híbrido

---

# 🚀 Características da Estratégia Distribuída

- 🏢 Região dedicada no cliente
- 🔗 Integração Multicloud (AWS, Azure, Google)
- 🗄️ Banco de dados Oracle nativo
- ⚖️ Foco em compliance
- 🏛️ Ideal para setor público e empresas reguladas

# ❓ Questão de Simulado

## Pergunta:

Qual capacidade pode ser usada para proteger contra falhas dentro de um domínio de disponibilidade da OCI?

- Balanceador de carga
- Regiões
- Compartimentos
- ✅ Domínio de falha

---

# ✅ Resposta Correta: Domínio de Falha

## 📌 Por quê?

O Fault Domain fica dentro do Availability Domain e protege contra:

- 🖥️ Falhas de hardware
- 🔧 Manutenções
- ⚡ Problemas elétricos locais

### Exemplo:

Imagine 3 computadores separados:

- 💻 Computador 1 falha
- 💻 Computador 2 falha
- 💻 Computador 3 continua funcionando

Assim o serviço permanece ativo.

---

# 🧠 Resumo Final

A OCI oferece diferentes modelos de nuvem para atender compliance, desempenho e flexibilidade. O Fault Domain é essencial para alta disponibilidade e tolerância a falhas dentro da infraestrutura Oracle.
